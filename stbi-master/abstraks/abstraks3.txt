Telepon seluler atau ponsel kini tidak hanya sebagai alat komunikasi saja, namun juga sebagai media hiburan seperti memainkan game ataupun aplikasi. Dari fungsi tersebut dalam upaya membantu seseorang untuk mengukur tingkat kecerdasan serta mengenal sifat pada dirinya diperlukan suatu aplikasi tes Intelligence Quotient ( IQ ) dan tes kepribadian. Aplikasi tes IQ dan tes kepribadian ini dibuat sebagai bentuk baru dalam pelaksanaan pengukuran kecerdasan dan kepribadian yang bersifat mobile, sehingga memudahkan para pengguna karena dapat dilakukan kapan pun diperlukan dan di mana pun berada tidak terbatasi oleh ruang dan waktu. Aplikasi ini dibuat dengan menggunakan bahasa pemrograman Java 2 Micro Edition ( J2ME ) dan NetBeans IDE 6.8. Dengan adanya aplikasi tes IQ dan tes kepribadian ini terbentuklah suatu fasilitas baru, di mana pengguna dapat melakukan tes IQ dan tes kepribadian melalui media handphone dan diharapkan aplikasi ini dapat digunakan sebagai referensi.
